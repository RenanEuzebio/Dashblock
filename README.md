# Dashblock

## How to run

### 1. Clone repo
```bash
git clone https://github.com/RenanEuzebio/Dashblock.git
cd Dashblock
```

### 2. Install dependencies
```bash
uv install
```

### 3. Run backend
```bash
uv run litestar run
```

### 4. Run frontend
```bash
cd frontend
python -m http.server 8001
```
